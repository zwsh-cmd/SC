# ScriptMind - 編劇靈感筆記本

這是一個專為編劇設計的單頁應用程式 (Single Page
Application)，無需安裝，直接瀏覽器開啟即可使用。

## 特色功能

1.  **結構化知識庫**：無限層級的分類樹狀圖，像 OneNote
    一樣管理您的編劇理論（三幕劇、救貓咪...）。
2.  **專用筆記格式**：
    -   大標題 (50字限制)
    -   說明 (500字限制，支援 Markdown)
    -   回應 (500字限制，支援多條回應)
    -   自動記錄最後編輯時間
3.  **每日靈感推播**：
    -   每次開啟 App 隨機推播一條筆記。
    -   智慧演算法確保 100 次內不重複。
4.  **收藏與歷史**：
    -   收藏您喜歡的筆記。
    -   查看最近 50 條推播紀錄，並可補收藏。

## 如何使用

1.  下載 `index.html` 與 `main.js` 到同一個資料夾。
2.  直接雙擊 `index.html` 開啟。
3.  所有的資料都儲存在您的瀏覽器 `localStorage` 中。
    -   **注意**：清除瀏覽器快取可能會導致資料遺失，建議定期使用 App
        內的「備份」功能。

## Markdown 語法支援

-   **粗體**： `**文字**`

-   *斜體*： `*文字*`

-   # 標題： `# 標題`

-   列表： `- 項目`

-   引用： `> 文字`
